# Value Creation — Decomposition & Planning Framework

## Purpose

Decompose PE investment returns into their operational and financial components, track progress against value creation plans, and build forward-looking value creation roadmaps.

## Value Creation Bridge

### Standard Decomposition

Total value change is decomposed into four primary drivers:

| Component | Formula | What It Measures |
|-----------|---------|-----------------|
| **Revenue Growth** | (Current Rev − Entry Rev) × Entry Margin × Entry Multiple | Value from top-line growth |
| **Margin Expansion** | Entry Rev × (Current Margin − Entry Margin) × Entry Multiple | Value from operational efficiency |
| **Multiple Expansion** | Current EBITDA × (Current Multiple − Entry Multiple) | Value from market re-rating |
| **Leverage Effect** | Net Debt Paydown × (1 − Implied Tax Rate) | Value from deleveraging |

**Total Value Created** = Sum of all components

### Reconciliation Check

The bridge MUST reconcile:
- Entry Equity Value + Total Value Created = Current Equity Value (± rounding)
- If it doesn't reconcile, identify the residual and label it

### Revenue Growth Sub-Decomposition

| Sub-Driver | Measurement |
|-----------|-------------|
| Organic pricing | Price increases × prior-year volume |
| Organic volume | Volume growth × prior-year price |
| New products/markets | Revenue from products/markets launched post-entry |
| Add-on acquisitions | Revenue from acquired businesses |
| FX impact | Currency translation effect |

### Margin Expansion Sub-Decomposition

| Sub-Driver | Measurement |
|-----------|-------------|
| Gross margin improvement | COGS reduction as % of revenue |
| SG&A optimization | SG&A reduction as % of revenue |
| Revenue mix shift | Impact of higher-margin product/service mix |
| Scale efficiencies | Fixed cost absorption from revenue growth |
| One-time items | Non-recurring costs removed from adjusted EBITDA |

## Value Creation Plan (Forward-Looking)

### Initiative Framework

Each value creation initiative should be documented as:

| Field | Description |
|-------|-------------|
| Initiative name | Clear, specific label |
| Category | Revenue growth / Margin improvement / Multiple expansion / Leverage |
| Description | What specifically will be done |
| Quantified impact | EBITDA impact ($M) and timing |
| Assumptions | Key assumptions underlying the estimate |
| Owner | Named individual responsible |
| Milestones | Key checkpoints with dates |
| Risk to plan | What could prevent execution |
| Dependencies | Other initiatives or external factors required |

## Benchmarking

Compare value creation profile against:

| Benchmark | Source | What to Compare |
|-----------|--------|----------------|
| Fund-level targets | IC memo / underwriting | Actual vs. plan by component |
| Prior fund portfolio | Internal data | Component mix, execution rate |
| Industry (Bain PE Report) | Annual publication | Revenue growth, margin expansion ranges |
| Peer funds (if available) | PitchBook, Cambridge | Quartile ranking by value creation driver |

## Critical Rules

1. Bridge components MUST reconcile to total value change
2. Entry metrics must match original IC memo / underwriting
3. Current metrics must match latest reported/audited figures
4. Specify multiple basis (EV/EBITDA, EV/Revenue, etc.) consistently
5. Separate operational value creation from market/multiple tailwinds
6. Forward plan assumptions must be explicitly stated and testable
7. Don't conflate organic growth with acquisition-driven growth
